# 📊 Credit Risk Data Analysis

## 📌 Project Overview

This project presents an exploratory data analysis (EDA) and data preprocessing study of a credit risk dataset.  
The objective is to understand borrower characteristics, identify data quality issues, and prepare the dataset for reliable financial analysis.

This study focuses on data exploration and cleaning rather than predictive modeling.

---

## 🎯 Objectives

- Analyze the structure of the credit risk dataset
- Identify and handle missing values
- Perform univariate, bivariate, and multivariate analysis
- Generate statistical summaries and visualizations
- Understand relationships between financial variables

---

## 📂 Dataset Description

The dataset contains borrower-level financial and demographic information, including:

- Age  
- Income  
- Employment length  
- Loan amount  
- Interest rate  
- Loan status  

These variables help describe borrower financial behavior and credit characteristics.

---

## 🔎 Exploratory Data Analysis

### 1️⃣ Data Understanding
- Checked dataset shape (rows and columns)
- Examined data types
- Generated descriptive statistics

### 2️⃣ Data Cleaning
- Identified missing values
- Calculated missing value percentages
- Applied median imputation for numerical features
- Verified cleaned dataset

### 3️⃣ Univariate Analysis
- Distribution of borrower age
- Income distribution
- Loan amount distribution
- Interest rate trends

### 4️⃣ Bivariate Analysis
- Income vs Loan Amount
- Employment Length vs Loan Amount
- Interest Rate vs Loan Status

### 5️⃣ Multivariate Analysis
- Correlation matrix
- Heatmap visualization
- Identification of relationships between numerical variables
