# Credit Risk Prediction Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) and preprocessing on a credit risk dataset.

## Objectives
- Analyze borrower characteristics
- Handle missing values
- Prepare data for modeling

## Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Dataset
Credit risk dataset used for financial risk assessment.
